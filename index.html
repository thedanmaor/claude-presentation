What Is a Token?

definition

~¾ of a word on average.
"Hello world" = 2 tokens.
"Supercharged" = 3 tokens.

what counts

Input tokens: your prompt + context
Output tokens: Claude's response
Output costs ~3-5× more than input

context window

200,000 tokens max per request.
= ~150,000 words
= a ~500-page novel
= a medium-sized codebase

why it matters

Pricing is per token.
Larger context = more tokens = higher cost.
Smart routing saves real money.

Three Tiers

Haiku — Fast

Cheapest. Fastest response.
Best for: simple lookups, renames, formatting, git ops, repetitive tasks.

Sonnet — Balanced

Best cost/capability ratio.
Best for: feature work, debugging, code writing, tests, most daily tasks.

Opus — Deep

Most capable. Slowest, priciest.
Best for: architecture decisions, complex refactors, multi-system reasoning.
